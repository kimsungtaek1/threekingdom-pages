# iOS App Store Connect URL 설정

GitHub Pages를 `main` 브랜치의 `/docs` 폴더로 배포하면 아래 URL을 바로 사용할 수 있습니다.

- 마케팅 URL: `https://kimsungtaek1.github.io/threekingdom/`
- 지원 URL: `https://kimsungtaek1.github.io/threekingdom/support.html`
- (선택) 개인정보 처리방침 URL: `https://kimsungtaek1.github.io/threekingdom/privacy.html`

## GitHub Pages 설정 방법

1. GitHub 저장소 `Settings`로 이동
2. `Pages` 메뉴 진입
3. `Source`를 `Deploy from a branch`로 설정
4. `Branch`를 `main`, 폴더를 `/docs`로 선택 후 저장
5. 1~3분 후 위 URL 접속 확인

## 포함된 파일

- `docs/index.html` (마케팅 페이지)
- `docs/support.html` (지원 페이지)
- `docs/privacy.html` (개인정보 처리방침)
- `docs/styles.css` (공통 스타일)
- `docs/assets/img/*` (페이지 이미지)
